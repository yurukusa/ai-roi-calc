# AI ROI Calculator

**Is AI actually worth it?** Enter your task details, see the real numbers across 28 models.

https://yurukusa.github.io/ai-roi-calc/

## What it does

Compares the cost of using AI vs hiring humans for specific tasks. Shows:

- **Per-task cost** for each of 28 AI models
- **Annual savings** (or losses) vs human labor
- **ROI percentage** for each model
- **Quick presets** for common use cases (code review, writing, support, etc.)

## Inputs

| Field | Description |
|-------|-------------|
| Task type | Code review, writing, analysis, support, translation, summarization |
| Tasks per day | How many times you do this task daily |
| Human hourly rate | Fully loaded cost (salary + benefits + overhead) |
| Minutes per task | How long it takes a human |
| Input/output tokens | Average token counts per task |

## Features

- 6 ready-made presets (Code Review, Blog Writing, Data Analysis, Support Tickets, Translation, Doc Summary)
- 28 models from 9 providers (Anthropic, OpenAI, Google, Meta, DeepSeek, Mistral, xAI, Cohere, Microsoft)
- Single HTML file, zero dependencies, works offline
- Honest disclaimer about limitations (quality, setup costs, review needs)
